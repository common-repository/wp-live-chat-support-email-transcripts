== Description ==

Allows agents to email chat transcripts during and after a live chat session.

> **Please note**
> This is an add-on for [WP Live Chat Support](https://wordpress.org/plugins/wp-live-chat-support/) and [WP Live Chat Support Pro](http://wp-livechat.com/)

Once installed, this add-on will enable a new set of options to WP Live Chat Support, as well as a button in the live chat window that allows you to email the chat transcript during and after each chat session.

== Installation ==

1. Please ensure you first have WP Live Chat Support installed and activated
2. Activate WP Live Chat Support - Email Transcripts
3. Settings are available in the Live Chat -> Settings page.
4. A button will become available in the live chat window which enables you to email the chat transcript to the user you are chatting with
